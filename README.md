# 🛒 APNA MART - Grocery Delivery Web Application

APNA MART is a modern and responsive grocery shopping web application built using **HTML, CSS, and JavaScript**. The project simulates a real-world online grocery delivery platform where users can browse products, manage their cart, apply coupons, track orders, and enjoy a complete e-commerce experience.

## 🚀 Live Demo

https://sanjeetkumar989159-hub.github.io/APNA_mart-Grocery-/
